## 최고의 집합 만들기
---
- 자연수 n개로 이루어진 집합의 원소의 합은 항상 s여야 함
- 위의 조건을 만족하는 집합 중 모든 원소의 곱이 최댓값을 기록하는 집합을 반환

```python
def solution(n, s):
    if n > s:
        answer = [-1]
    else:
        p, q = divmod(s, n) # p는 몫, q는 나머지
        answer=[p]*n
    
        for i in range(q):
            answer[i] += 1
        
        answer = sorted(answer)
    return answer
```
